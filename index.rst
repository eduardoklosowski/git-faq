Git FAQ
=======

Este texto contem problemas e perguntas sobre git, apresentando algumas dicas de como os mesmos poderiam ser resolvidos ou sugestões de como tratá-los.

Este não é um texto finalizado, recebendo atualizações com novos tópicos conforme as ideias forem surgindo e os tópicos escritos. Caso tenha alguma dica ou sugestão, abra uma issue no `repositório do GitHub <https://github.com/eduardoklosowski/git-faq>`_, mande seu texto como pull request, ou entre em contato pelo grupo `Git Brasil do Telegram <https://t.me/GitBrasil>`_.
